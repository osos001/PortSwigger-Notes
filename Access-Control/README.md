# Access Control

These are my personal notes while studying the Access Control topic from PortSwigger Web Security Academy.

## Topics Covered

- Insecure Direct Object Reference (IDOR)
- Horizontal Privilege Escalation
- Vertical Privilege Escalation
- URL-based Access Control
- Method-based Access Control
- Referer-based Access Control
- Multi-step Process Vulnerabilities

## What I Learned

- Never trust client-side authorization.
- Always enforce authorization checks on the server.
- Small logic flaws can lead to privilege escalation.
